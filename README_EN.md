## 📌 Curated Links

### 🌐 System / Infra
* **[Build Your Own Redis](https://build-your-own.org/redis/)** - A comprehensive guide to building core Redis functionalities from scratch using C/C++ or Go without external libraries. Ideal for learning Network Sockets, Protocol Parsing, and Event Loops.
  * `Tags` : `Network` `Event Loop` `C/C++` `Go` `Low-Level`
* **[System Design Notes](https://github.com/liquidslr/system-design-notes)** - A well-structured summary of major architectural patterns (Load Balancing, Caching, DB Sharding, etc.) used in large-scale distributed systems.
  * `Tags` : `Architecture` `Distributed System` `Backend`
* **[Practice C Programming Projects](https://github.com/labex-labs/practice-c-programming-projects)** - A collection of practical C programming projects designed to strengthen low-level control fundamentals, including pointer operations and memory allocation.
  * `Tags` : `C Language` `Memory Allocation` `Pointers`
* **[Computer Networking Fundamentals](https://labs.iximiuz.com/courses/computer-networking-fundamentals)** - An interactive networking practical course based on Linux virtual environments (Containers, Namespaces) to verify routing and TCP/IP flows.
  * `Tags` : `Linux` `Networking` `TCP/IP` `Interactive`

### 🤖 AI / ML & Core Mathematics
* **[Math Behind Cross-Entropy Loss](https://outcomeschool.com/blog/math-behind-cross-entropy-loss)** - A deep dive into the mathematical derivation of Cross-Entropy Loss from an information theory perspective, which forms the basis of classification and contrastive learning.
  * `Tags` : `Information Theory` `Loss Function` `Math`
* **[Paper Reading (Mu Li)](https://github.com/mli/paper-reading)** - A repository of deep learning landmark paper analyses by Dr. Mu Li, former Principal Scientist at AWS.
  * `Tags` : `Deep Learning` `Paper Reading` `Landmark Models`
* **[Transformer Explainer](https://github.com/poloclub/transformer-explainer)** - An interactive 3D visualization tool illustrating tensor operations inside the Transformer architecture (based on GPT-2).
  * `Tags` : `Transformer` `Visualization` `GPT-2` `Web Tool`
* **[Stanford CS229 Machine Learning Cheatsheets](https://github.com/afshinea/stanford-cs-229-machine-learning)** - VIP cheatsheets and refreshers for Stanford's CS229 course, covering supervised/unsupervised learning, deep learning formulas, and probability/linear algebra basics.
  * `Tags` : `Stanford CS229` `Cheatsheet` `Linear Algebra`
* **[Sophon AI Research Papers](https://sophon.at/papers)** - A discovery platform for tracking real-time trending AI research papers, models, and benchmarks across fields like language modeling and agents.
  * `Tags` : `AI Trend` `Research Paper` `LLM` `Benchmark`

### 🛠 Tools & AI Productivity

## 📚 Books & Lecture Notes (Official Source Links)

### 💻 Operating Systems & Low-Level
* **[MIT xv6 RISC-V Source & Commentary Book](https://pdos.csail.mit.edu/6.1810/2025/xv6/)**
  * `Replaces` : `book-riscv-rev5.pdf` / `xv6-src-booklet-rev5.pdf`
  * Official textbook and complete C/Assembly source booklet for the xv6 Unix-like kernel used in MIT's operating systems course.
  * `Type` : `University Textbook` `Operating System` `RISC-V`

### 🤖 AI Engineering & Prompting
* **[Andrew Ng's Full AI Prompting Course](https://www.youtube.com/watch?v=8ib4Qnh2HFE)** / **[DeepLearning.AI Official](https://www.deeplearning.ai/courses/ai-prompting-for-everyone)**
  * `Replaces` : `Full-AI-Prompting-Course-with-Andrew-Ng-ko.pdf`
  * Advanced prompting engineering course taught by Professor Andrew Ng, focusing on programmatic context management and multi-agent pipeline designs.
  * `Type` : `Video Course` `Prompt Engineering` `DeepLearning.AI`

### 📐 Mathematics & Core Algorithms
* **[Mathematical Foundations of Machine Learning (Seongjai Kim)](https://www.youtube.com/@mathtalent)**
  * `Replaces` : `Machine_Learning_Lecture.pdf`
  * Lecture materials presenting rigorous mathematical derivations of ML algorithms, including SVM KKT conditions, PCA/SVD matrix decompositions, and clustering sufficency proofs.
  * `Type` : `Lecture Notes` `Machine Learning Math` `Matrix Decomposition`
* **[Calculus: Early Transcendentals Web Text](https://open.umn.edu/opentextbooks/textbooks/415)**
  * An open calculus textbook by Professor David Guichard listed in the University of Minnesota Open Textbook Library. Highly relevant for self-studying optimization algorithms like Gradient Descent.
  * `Type` : `Open Textbook` `Calculus` `Optimization`
* **[Machine Learning Systems (Harvard Edge / MIT Press)](https://github.com/harvard-edge/cs249r_book)**
  * `Replaces` : `Machine-Learning-Systems.pdf`
  * A premium open-source book used at Harvard and MIT covering everything from single-node deployment optimizations to large-scale distributed model infrastructure.
  * `Type` : `Open Book` `MLSys` `Infrastructure`
* **[Breaking the Sorting Barrier for Directed SSSP (arXiv)](https://arxiv.org/abs/2504.17033)**
  * `Replaces` : `Breaking the Sorting Barrier...pdf`
  * A groundbreaking research paper introducing a deterministic algorithm that breaks Dijkstra's traditional sorting complexity bottleneck for directed Single-Source Shortest Paths.
  * `Type` : `Research Paper` `Graph Algorithm` `arXiv`
