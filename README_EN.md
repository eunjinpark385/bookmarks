## 📌 Links

### System / Infra
* [Build Your Own Redis](https://build-your-own.org/redis/) - A step-by-step guide to building core Redis features from scratch using C/C++ or Go without external libraries. Useful for learning Network Sockets, Protocol Parsing, and Event Loop architectures.
* [System Design Notes](https://github.com/liquidslr/system-design-notes) - A comprehensive summary of key architecture patterns for large-scale system design, covering Load Balancing, Caching, DB Sharding, and Replication.
* [Practice C Programming Projects](https://github.com/labex-labs/practice-c-programming-projects) - A collection of practical C programming projects with step-by-step guidelines to enhance low-level control skills such as pointer arithmetic and memory allocation.
* [Computer Networking Fundamentals](https://labs.iximiuz.com/courses/computer-networking-fundamentals) - An interactive networking course based on a Linux virtual environment (Containers, Namespaces) to test and verify Routing tables and TCP/IP flows.

### AI / ML
* [Math Behind Cross-Entropy Loss](https://outcomeschool.com/blog/math-behind-cross-entropy-loss) - An in-depth explanation of deriving Cross-Entropy Loss from an information theory perspective (Entropy, KL-Divergence), which forms the foundation of classification and contrastive learning.
* [Paper Reading (Mu Li)](https://github.com/mli/paper-reading) - A video lecture and analysis archive of landmark deep learning papers by Dr. Mu Li (former Principal Scientist at AWS).
* [Transformer Explainer](https://github.com/poloclub/transformer-explainer) - A 3D interactive visualization tool running in the browser to map tensor operations and data flows inside the Transformer (GPT-2 based) architecture.

### Visualization / Utility
* [Manim Community](https://github.com/ManimCommunity/manim) - A mathematical animation engine to programmatically animate concepts or algorithmic mechanisms using Python. Useful for creating high-quality graphics/GIFs for presentations.
* [VisuAlgo](https://visualgo.net/en) - A visualization platform demonstrating the step-by-step behavior of core data structures and algorithms coupled with pseudo-code tracking.
* [CSVistool (ArrayList)](https://csvistool.com/ArrayList) - A visualization tool for the inner mechanisms of an ArrayList, showcasing data shifting during insertion/deletion and internal resizing processes.
* [cobalt.tools](https://cobalt.tools) - A clean and lightweight web utility to extract media files without tracking scripts or intrusive ads. Useful for gathering dataset sources.

### Writing / Review
* [Google Engineering Practices](https://github.com/google/eng-practices/blob/master/review/reviewer/comments.md) - Google's internal code review guidelines for learning professional and constructive communication when giving technical feedback.
* [Academic Phrasebank](https://www.phrasebank.manchester.ac.uk/) - An academic writing corpus provided by the University of Manchester, categorizing formal sentence structures for each section of a research paper (Intro, Methodology, Results, etc.).
* [Write & Improve](https://writeandimprove.com/) - An AI-powered automated feedback service developed by Cambridge to evaluate English writing drafts, catch grammatical errors, and provide real-time suggestions.
